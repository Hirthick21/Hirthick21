### 🚀 Featured Projects

#### 🤖 [forex-ai-bot](https://github.com/Hirthick21/forex-ai-bot)
Autonomous trading system using **ICT / Smart Money Concepts** methodology with a multi-timeframe analysis cascade (Daily → 4H → 1H → 15m → 5m). **GPT-4o** acts as the live decision-making agent evaluating trade setups, with **MetaTrader 5 (MT5)** integration for execution. Currently running on a live funded evaluation account and being extended to support crypto pairs (BTC/USD, ETH/USD).

`Python` `GPT-4o` `MT5` `Algo Trading`

#### 🧠 [Vibe-Trading](https://github.com/Hirthick21/Vibe-Trading)
Open-source AI trading research platform that orchestrates **OpenRouter** and **DeepSeek** models to automate market research and analysis workflows. Deployed on Render.com.

`Python` `OpenRouter` `DeepSeek` `AI Agents`
